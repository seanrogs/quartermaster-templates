What are qmtemplates?

qmtemplates are faction template files for use with Quartermaster.

All faction templates are made and contributed by the Quartermaster community. Have one to contribute? Email it to contribute@quartermaster.app

Don't see what you're looking for?

Get involved! You can create your own faction templates in Quartermaster itself. Find out more about editing a Quartermaster template in the Quartermaster User Guide.

Having trouble downloading?

Open this page in Safari on your device for the best results. Google Chrome, the Facebook browser, and other in-app browsers can't download files on iOS.

Make sure you have the latest game system installed for the template. Game system files are listed with the faction templates for each game system.
